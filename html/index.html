Q7

If we do the shallow copy, these pointers simply point to the same object. In this case, if we delete one pointer, the pointed object is destructed according to the destructor. Thus, the other pointer point to an address which contains unknown data. To solve this problem, we need to overload the operator =. We should do the deep copy. That is to say, when copying pointers, we create a new copy to the pointed object and make the pointer point to it.
